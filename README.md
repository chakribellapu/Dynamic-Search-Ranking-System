\# 🔍 Dynamic Search Ranking System



\## 📌 Project Overview

Dynamic Search Ranking System is a Java-based Command Line Interface (CLI) application that ranks search results dynamically using a weighted scoring formula. The system adjusts rankings based on keyword relevance, freshness, and user engagement.



---



\## 🚀 Features

\- ✅ Weighted ranking algorithm

\- ✅ Priority Queue (Max Heap) for top-k results

\- ✅ Real-time engagement simulation

\- ✅ Command Line Interface (CLI)

\- ✅ Object-Oriented Design



---



\## 🛠 Technologies Used

\- Java

\- OOP Concepts

\- Data Structures (HashMap, PriorityQueue)

\- CLI-based interaction



---



\## 🧠 Ranking Formula

Final Score is calculated using:



Final Score =  

0.6 × Keyword Match  

\+ 0.25 × Freshness  

\+ 0.15 × Engagement  



This ensures relevant and fresh content ranks higher while user engagement dynamically improves ranking.
